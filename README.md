## NLP Porn Detection
Participated in Kaggle's Porn Detection [competition](https://www.kaggle.com/competitions/ml-2024-fall-porn-detection/overview) (HSE & VK Deep Learning Course). Built and fine-tuned NLP classifiers for explicit content detection on multimodal text data. 

#### Contained files:
* `eda_porn_detection.ipynb` - Exploratory Data Analysis (preprocessing, some dataset statistics)
* `fasttext_porn_detection.ipynb` - FastText baseline classifier
* `bert_porn_detection.ipynb` - Fine-tuned DeBERTa model for optimized performance

#### Results:
Achieved accuracy: **0.9664**
